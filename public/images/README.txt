IMAGE FILES — How to add your own
==================================

Drop JPG/PNG files into the appropriate subfolder. Defaults reference:

images/kuwait/   →  kuwait-01.jpg … kuwait-10.jpg

Each Kuwait Childhood question has an optional "image" field you can
edit in index.html. If the file is missing, the question still works
— the text alone is enough to play.

  { q: "Recognize this Kuwaiti landmark?",
    image: "images/kuwait/kuwait-01.jpg",
    options: [...], answer: 0 }

For best results, use clear, recognizable images at ~600px wide.
